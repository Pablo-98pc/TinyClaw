Data Flow

  1. User types a message in the SwiftUI chat interface
  2. Dispatcher (BERT-tiny, always in memory) classifies intent in <50ms
  3. Model Manager checks if the target specialist is loaded; if not, loads it (evicting LRU model if needed)
  4. Specialist runs inference via Core ML on the Neural Engine
  5. Response streams back to the chat UI with a specialist badge (e.g., "Code" or "Chat")
